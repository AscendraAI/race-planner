Race Planner parameter reference

These URLs prefill tool inputs. They never auto-run calculations, so analytics stay tied to user clicks.

GPX Race Plan
ParamFormatBehavior
goalH:MM:SSPrefills goal finish time.
temp25Parsed as Celsius for AI links; invalid values are ignored.
level1, 2, 3 or 30, 45, 60Prefills fueling level.
racestringReserved for future race presets. Parsed but unused.
langko or enForces page language.
https://raceplanner.online/gpx/?goal=4:30:00&temp=25&level=2
